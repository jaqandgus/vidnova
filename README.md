## Про проєкт

**Vidnova** — інструмент для верифікованих НГО, благодійних фондів та громад України, який дозволяє залучати міжнародні донати у стейблкоїнах із on-chain прозорістю кожної транзакції та автоматизованою звітністю про використання коштів.

### Проблема

Мільярди доларів донорських коштів направляються на відновлення України, проте:

- донори не бачать, куди саме йдуть кошти після переказу
- отримувачі не завжди верифіковані, що породжує ризики шахрайства
- звітність — фрагментована, паперова або відсутня
- малі НГО не мають інструментів для прямого збору коштів на міжнародних ринках

Це породжує донорську втому та обмежує приплив зовнішнього фінансування у відновлення.

### Рішення

Платформа, де кожен проєкт відновлення представлений смартконтрактом-ескроу. Кошти від донорів фіксуються on-chain, розблоковуються поетапно після підтвердження виконання робіт, а вся звітність публічно доступна для верифікації.

```
[Донор] → [USDT через смартконтракт] → [Ескроу] → [Етап виконано + звіт] → [Виплата НГО]
                                            ↓
                                   [Публічний дашборд]
```

## Ключові механіки

| Компонент | Опис |
|-----------|------|
| **Ескроу-контракт** | Кошти заморожуються до підтвердження виконання етапу. Розблокування — через мультипідпис (НГО + незалежний верифікатор). |
| **Верифікація НГО** | DID-атестація організацій. Тільки верифіковані учасники можуть створювати проєкти зборів. |
| **SBT-сертифікати** | Кожен донор отримує soulbound NFT як підтвердження участі — для CSR-звітності корпоративних донорів. |
| **IPFS-звітність** | Документи, фото та акти прикріплюються до on-chain записів через децентралізоване зберігання. |
| **Публічний реєстр** | Усі транзакції індексуються через The Graph і доступні в реальному часі. |

## Технологічний стек

**Smart contracts**
- Solidity ^0.8.20
- OpenZeppelin contracts (security primitives)
- Foundry (testing framework)

**Frontend**
- React 18 + Vite
- TypeScript
- wagmi + viem (Web3 integration)
- TailwindCSS-подібні стилі

**Інфраструктура**
- BNB Chain (primary network)
- IPFS / Pinata (decentralized storage)
- The Graph (event indexing)
- Vercel / GitHub Pages (deployment)

**Wallets**
- WalletConnect, MetaMask, Binance Wallet, Coinbase Wallet

## Локальний запуск

```bash
git clone https://github.com/jaqandgus/vidnova.git
cd vidnova
npm install
npm run dev
```

Відкрити http://localhost:5173

## Деплой

```bash
npm run deploy
```

Деплой автоматично публікує збірку на GitHub Pages.
